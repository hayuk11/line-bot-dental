PASSOS RÁPIDOS:

1. Instale as dependências:
   pip install -r requirements.txt

2. Edite app.py e substitua SEU_ACCESS_TOKEN pelo seu token da LINE Messaging API.

3. Rode localmente com:
   python app.py

4. Para produção (Render.com):
   - Set Environment: Python 3
   - Start Command: gunicorn app:app
